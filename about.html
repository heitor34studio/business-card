Sobre Mim!

Olá, me chamo Heitor Dutra Bento. Tenho 18 anos e sou Dev Full-Stack. Atualmente estudo Análise e Desenvolvimento de Sistemas na Modalidade EAD na Universidade da Estácio. Sou fluente em Português e Inglês e Intermediário em Japonês. Minha jornada nessa área começa quando fiz o Curso completo de Desenvolvimento Web 2018 durante a quarentena em 2020. Em 2021 conheci o WordPress e comecei a praticar a criação de sites nele. Clique aqui para ver mais detalhes dos seguintes projetos:
Meu 1° PROJETO foi um site institucional para o negócio da minha mãe. Ainda em 2021 recebi uma mesa digitalizadora, que usei para fazer alguns designs para os próximos projetos e para desenhar mesmo.
Meu 2° PROJETO foi meu portfólio de projetos.
Meu 3° PROJETO agora voltando para sites programados não WordPress, foi o ThonThon. Uma adaptação do projeto básico "Twitter Clone" do curso de 2020, porém com muitas funcionalidades a mais.
Meu 4° PROJETO foi o VavaLineups, ele focava em mostrar jogadas para se fazer com a personagem Viper do jogo Valorant. Nele quis implementar formas de pagamento para a criação de uma conta VIP na plataforma.
Meu 5° PROJETO foi o BitPrice, uma plataforma que checa os atuais preços das criptomoedas em API's de diversas corretoras.
Meu 6° PROJETO foi o Renthon Pizzaiolo, um sistema completo de Pizzaria em React, simulando a criação dos produtos, os pedidos na cozinha, e o aplicativo do garçom no celular.
Meu 7° PROJETO foi esse Cartão de Visitas, portfolio que fiz com os cursos da Alura, fornecidos pelo Programa ONE - Oracle Next Education.
